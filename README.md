# CyberVonX Labs

This repository contains my pentesting labs, reports, and scripts. All labs were performed locally using Kali Linux and vulnerable VMs.

## Repo Structure

cybervonx-labs/
├── dvwa/ # Web application testing
├── metasploitable/ # Network and service exploitation
├── vulnhub/ # Custom VM challenges
├── reports/ # Lab reports
└── scripts/ # Custom scripts/tools

## Usage
1. Start your vulnerable VM.
2. Launch Kali Linux.
3. Perform testing following the lab notes inside each directory.
4. Reports contain step-by-step documentation.

## Notes
- All testing performed locally.
